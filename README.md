## **📘 Web Engineering – Weekly Breakdown (16 Weeks)**

### [Week 1: Introduction to Web & Web Engineering](markdown/lecture1.md)

* What is the Web? What it is *not*
* Web vs Internet
* Evolution of the Web (Web 1.0 → 3.0)
* What is Web Engineering? How it differs from Web Technologies
* Overview of client-server architecture

---

### [Week 2: How the Web Works (Deep Dive)](markdown/lecture2.md)

* HTTP/HTTPS basics (request/response)
* URL structure
* DNS resolution (brief)
* Role of browsers
* Introduction to developer tools (Network tab)

👉 *Activity:* Inspect a real webpage request using DevTools

---

### [Week 3: HTML Fundamentals](markdown/lecture3.md)

* Structure of HTML document
* Tags, elements, attributes
* Headings, paragraphs, links, images

---

### **Week 4: Advanced HTML**

* Lists, tables, forms
* Semantic HTML (header, footer, section, article)
* Accessibility basics

👉 *Lab:* Build a structured form (e.g., student registration)

---

### **Week 5: CSS Basics**

* Inline, internal, external CSS
* Selectors, colors, fonts
* Box model

👉 *Lab:* Style previous HTML pages

---

### **Week 6: CSS Layouts**

* Flexbox
* Grid
* Responsive design basics
* Media queries

👉 *Lab:* Convert static page into responsive layout

---

### **Week 7: JavaScript Basics**

* Role of JavaScript in web
* Variables, data types, operators
* Functions and events

👉 *Lab:* Add interactivity (button click, alerts)

---

### **Week 8: JavaScript & DOM Manipulation**

* DOM structure
* Selecting and modifying elements
* Event handling

👉 *Lab:* Dynamic form validation

---

### **Week 9: Midterm + Project Proposal**

* Midterm Exam
* Introduction to course project
* Idea discussion and approval

---

### **Week 10: Client-Side Engineering Concepts**

* Separation of concerns (HTML, CSS, JS)
* Code organization
* Intro to frameworks (conceptual only)

👉 *Activity:* Refactor messy code into structured format

---

### **Week 11: Server-Side Basics**

* What is server-side programming?
* Request handling concept
* Introduction to Node.js / PHP (choose one)
* Static vs dynamic pages

👉 *Lab:* Simple server returning HTML

---

### **Week 12: Working with Data**

* Forms and data submission
* GET vs POST
* Intro to JSON
* Basic database concept (high level)

👉 *Lab:* Submit and display form data

---

### **Week 13: Web Architecture & Design**

* 3-tier architecture
* MVC pattern (conceptual)
* RESTful services (intro)

👉 *Activity:* Map a website to MVC components

---

### **Week 14: Web Security Basics**

* Common vulnerabilities (XSS, SQL Injection – concept only)
* Authentication vs Authorization
* HTTPS importance

👉 *Discussion:* Why security matters in real systems

### **Week 15: Testing, Deployment & Maintenance**

* Basic testing concepts
* Debugging techniques
* Deployment (shared hosting / GitHub Pages)
* Version control (Git intro)

👉 *Lab:* Deploy project

---

### **Week 16: Final Project Presentations**

* Project demonstrations
* Viva / evaluation
* Course wrap-up
